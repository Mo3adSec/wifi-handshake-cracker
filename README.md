
<h1 aling="left">GhostHnadshake</h1>
<h2 aling="left"> 📌 Description</h2>
  <p aling="left"> GhostHandshake is a tool designed to capture WPA/WPA2 handshakes and perform offline password cracking .</p>

<h2 aling="left"> 📌 Features</h2>
<p aling="left">- 📍 capture WPA/WPA2 handshake</p>
<p aling="left">- 📍 Detect connected clients</p>
<p aling="left">- 📍 Automatic deauthentication</p>
<p aling="left">- 📍 Offline passwoed cracking</p>


<h2 aling="left"> 🌐 installation</h2>

<pre aling="left">
 git clone https://github.com/Mo3adSec/wifi-handshake-cracker.git
 cd wifi-handshake-cracker
 python3 GhostHandshake.py</pre>
